Credits & References

A transparent breakdown of the resources, tooling, and generation involved in establishing this academic multimedia project.

Technical Toolbelt

  • Web Architecture: HTML5, CSS3, Vanilla JS
  • Frameworks: Bootstrap 5, Bootstrap Icons, Font Awesome
  • Motion Design: AOS (Animate On Scroll) Library
  • Media Display: GLightbox
  • Fonts: Google Fonts (Inter & Poppins)

AI Usage Disclosure

"The project used AI tools to generate selected media and draft content. Specifically, AI was used to create initial image concepts, draft explanatory text, and assist in narration production. All AI outputs were reviewed, edited, and adapted manually before final integration into the website. The exact AI-assisted elements are identified in the report and, where relevant, labeled inside the website interface."

Specific AI Inputs:
  • Concept Drafting: Text assisted by large language models.
  • Imagery: Base visual concepts generated by diffusion models, then retouched in Photo editing tools.
  • Audio: Explainer narrations generated via TTS (Text-to-Speech) platforms.

Developed as part of coursework exploring multimedia pipelines. This site demonstrates the integration of static web standards alongside dynamically synthesized assets.
